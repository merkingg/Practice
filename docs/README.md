## 3. Создание статического сайта

**Время:** 18 часов

Разработан многостраничный сайт проекта **«Книга Будущего»** на HTML и CSS.

| Страница | Описание |
|---|---|
| `index.html` | Главная — аннотация проекта, актуальность, цели |
| `about.html` | О проекте |
| `team.html` | Участники и личный вклад |
| `journal.html` | Журнал прогресса |

Использованы CSS Grid, Flexbox, hover-эффекты. Сайт оформлен графическими материалами и иллюстрациями.
